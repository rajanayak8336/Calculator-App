# 📖 Project Description

The **Calculator App** is a frontend-based web application designed to perform basic mathematical calculations such as addition, subtraction, multiplication, and division.

The main objective of this project is to create a simple and interactive calculator with an attractive user interface. It demonstrates the practical implementation of HTML, CSS, and JavaScript concepts such as event handling, DOM manipulation, and user input management.

The application provides a responsive design that works efficiently on desktops, tablets, and mobile devices.

# 💻 Project Explanation

## 1. User Interface Design

The calculator interface contains:

* Display screen to show entered values and results
* Number buttons (0-9)
* Arithmetic operator buttons
* Clear button
* Equal button for displaying results

The layout is designed using CSS to provide a simple and attractive user experience.

---

## 2. Calculation Logic

JavaScript is used to handle all calculator operations.

The application:

* Takes input from users through button clicks.
* Stores the entered values.
* Performs required mathematical operations.
* Displays the final result on the calculator screen.

---

## 3. Button Functionality

Each button performs a specific action:

* Number buttons → Add numbers to display
* Operator buttons → Select mathematical operations
* Equal button → Calculate and display result
* Clear button → Reset calculator values

# 📱 Responsive Design

The calculator is designed to work smoothly on:
